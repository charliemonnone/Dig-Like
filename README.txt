Run Instructions:
	Run via the command(without quotes):
		"python mydig.py <input>"
	where <input> is a url, such as www.google.com, www.nytimes.com, etc.

CNAME Resolution:
	CNAME resolution is implemented and enabled, but can be toggled off by setting:
		"RESOLVE_CNAME = False"
	in mydig.py.
